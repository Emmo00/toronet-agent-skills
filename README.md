# ToroNet Agent Skills

A collection of installable AI agent skills for developers building on ToroNet.

## Skills

This repository will contain the following skills:

- `toronet-smart-contract-development`: writing and reviewing ToroNet-compatible smart contracts
- `toronet-smart-contract-deployment`: deploying contracts to ToroNet correctly
- `toronet-app-integrations`: frontend/backend integration guidance
- `toronet-sdk`: SDK usage patterns and implementation help

## Installation

Install from this repository using your preferred Agent Skills-compatible workflow.

Examples:

```bash
npx skills add emmo00/toronet-agent-skills
```

Or install a specific skill from this repository (syntax varies by installer):

```bash
npx skills add emmo00/toronet-agent-skills --skill toronet-smart-contract-deployment
```
